# 원로지 검색 등록 및 공유 미리보기 안내

- 홈페이지 주소: `https://wonlogics.co.kr`
- 대표 sitemap: `https://wonlogics.co.kr/sitemap.xml`
- robots.txt: `https://wonlogics.co.kr/robots.txt`
- 공유 대표 이미지: `https://wonlogics.co.kr/hero-global-network.webp`

## 1. Google Search Console 등록

1. [Google Search Console](https://search.google.com/search-console/about)에 접속한다.
2. `URL 접두어` 속성으로 `https://wonlogics.co.kr` 를 등록한다.
3. 안내되는 소유권 확인 절차를 완료한다.
4. 등록 후 속성 화면에서 `sitemaps` 메뉴로 이동한다.
5. `https://wonlogics.co.kr/sitemap.xml` 을 제출한다.
6. 메인 페이지 `https://wonlogics.co.kr` 를 URL 검사로 확인한 뒤 색인 요청을 진행한다.

## 2. 네이버 서치어드바이저 등록

1. [네이버 서치어드바이저](https://searchadvisor.naver.com/)에 로그인한다.
2. `웹마스터 도구`에서 `https://wonlogics.co.kr` 사이트를 등록한다.
3. 안내되는 소유 확인 절차를 완료한다.
4. 사이트맵 제출 메뉴에서 `https://wonlogics.co.kr/sitemap.xml` 을 제출한다.
5. 필요하면 URL 검사 또는 수집 요청 메뉴에서 메인 URL 수집 상태를 확인한다.

## 3. sitemap.xml 제출

- 구글 Search Console과 네이버 서치어드바이저 모두 `https://wonlogics.co.kr/sitemap.xml` 을 제출한다.
- `robots.txt` 에도 sitemap 경로가 포함되어 있어 검색엔진이 참고할 수 있다.

## 4. URL 수집 / 색인 요청

- 구글: Search Console의 URL 검사에서 `https://wonlogics.co.kr` 확인 후 색인 생성 요청
- 네이버: 서치어드바이저의 URL 검사 또는 수집 요청 기능에서 메인 URL 확인

## 5. 확인 방법

- `site:wonlogics.co.kr`
- `"원로지"` 검색

반영은 바로 되지 않을 수 있으며, 보통 며칠에서 몇 주까지 걸릴 수 있다.

## 6. 링크 공유 미리보기 확인

- 카카오톡에 `https://wonlogics.co.kr` 붙여넣기
- 슬랙, 문자, 카카오톡에서 title, description, image 가 정상 노출되는지 확인
- 일부 서비스는 기존 미리보기 캐시를 유지하므로 변경 직후에는 바로 바뀌지 않을 수 있다.

## 7. 공유 이미지 참고

- 현재 공유 대표 이미지는 `hero-global-network.webp` 를 사용한다.
- 공유 미리보기 품질을 더 높이려면 추후 1200x630 비율의 전용 OG 이미지를 별도로 준비해 교체하는 것을 권장한다.
